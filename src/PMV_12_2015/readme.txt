Je tourne sous un environment virtuel virtualenv (parceque va-t-en installer node sur les machines de l'école sans ca),
sur lequel j'ai installé et lancé un environnement virtuel node (nodeenv).

Sinon en gros, pour faire tourner le truc faut installer node, puis installer (npm c'est la package manager de node)
les package décrits dans le json. Si mes souvenirs sont bons, et si j'ai commit le bon truc, npm install si
lancé dans le dossier contenant le json devrait automatiquement installé les package listés dedans.


$pip install nodeenv

$npm install
